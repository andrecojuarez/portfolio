Tool

recon-runner

📅 02 Nov 2025 · 🐍 Python 3.11+ · 📦 GitHub
PythonReconAutomation

recon-runner orquesta Nmap, Gobuster y Nuclei contra un target, consolidando resultados en un reporte Markdown listo para entregar.

Instalación

git clone https://github.com/usuario/recon-runner
cd recon-runner
pip install -r requirements.txt

Uso

python recon.py --target 10.10.10.10 --out report.md

Opciones

  • --target — IP o dominio objetivo (requerido).
  • --out — Archivo de salida en Markdown.
  • --fast — Solo puertos top-1000.

Cómo funciona

El flujo interno es simple:

  1. Escaneo de puertos con Nmap (-sV -sC).
  2. Para cada puerto HTTP detectado, lanza Gobuster con wordlist configurable.
  3. Pasa las URLs descubiertas a Nuclei para detección de CVEs conocidos.
  4. Compila todo en un Markdown estructurado.

Úsalo solo contra objetivos para los que tengas autorización explícita.
